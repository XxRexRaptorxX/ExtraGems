## üìñ Description

ExtraGems is a mod that adds a few new gems for tools, weapons, armor, blocks and more!

The gems have stats similar to diamond, but a durability similar to iron and the highest enchantability, with slightly changes between its variants.

-----

#### Features:

- 6 materials! [<span style="color:green">emerald</span>, <span style="color:red">ruby</span>, <span style="color:blue">sapphire</span>, <span style="color:purple">amethyst</span>, <span style="color:gold">topaz</span>, <span style="color:lightblue">crystal</span>]
- amulets with cool side effects!
- can be used for beacons
- emissive textures support
- trimmable armor
- [Epic Fight](https://www.curseforge.com/minecraft/mc-mods/epic-fight-mod) support
- [JEED](https://www.curseforge.com/minecraft/mc-mods/just-enough-effect-descriptions-jeed) support
- [Energized Power](https://www.curseforge.com/minecraft/mc-mods/energized-power) support
- [Mekanism](https://www.curseforge.com/minecraft/mc-mods/mekanism) support
- config file
- update-checker

Do you want more ores? Try the [ExtraOres](https://www.curseforge.com/minecraft/mc-mods/extraores) mod! :D

## üóÉÔ∏è Dependencies

### Supported:
- **Just Enough Items (JEI)**
  - Provides some additional info pages and shows all recipes
  - [CurseForge][cf_jei] | [Modrinth][mr_jei]
- **Just Enough Effect Descriptions (JEED)**
  - Shows which items and blocks contain effects
  - [CurseForge][cf_jeed] | [Modrinth][mr_jeed]
- **MoreMcmeta Emissive Textures**
  - Enables the emissive textures of the mod
  - [CurseForge][cf_mcmeta] | [Modrinth][mr_mcmeta]
- **Continuity**
  - Enables the emissive textures of the mod
  - [CurseForge][cf_continuity] | [Modrinth][mr_continuity]
- **Shimmer**
  - Enables colored lightning 
  - [CurseForge][cf_shimmer] | [Modrinth][mr_shimmer]
- **Energized Power**
  - Crystal growth chamber recipes
  - [CurseForge][cf_ep] | [Modrinth][mr_ep]
- **Mekanism**
  - Enriching and nucleo synthesizing recipes 
  - [CurseForge][cf_mek] | [Modrinth][mr_mek]
- **Curios**
  - Provides additional accessory slots in which the items can be placed
  - [CurseForge][cf_curios] | [Modrinth][mr_curios]
